@[TOC](StoneMusic)
## 简要介绍：
- stoneMusic是一款基于Android的音乐播放器；
- 该项目采用MVP拆分模式，减少代码耦合。

## 历史记录

2018/7
:  采用MVC模式，实现本地音频播放器基本功能：音频播放和进度条控制

2019/4
:  采用MVP模式重构，并加入状态栏控制，后台播放，耳机插拔，音频焦点的占用和释放，加入歌控件，增加自动搜索歌词功能

2019/8
:  使用kotlin混合开发，重构结构
